# Star Coffee Shop

A modern, responsive coffee shop website built with HTML, CSS, and JavaScript. This project showcases a professional online presence for a coffee business with an elegant design and smooth user experience.

## Project Overview

Star Coffee Shop is a single-page application (SPA) that displays information about a coffee shop, its products, and customer reviews. The website features a responsive design that works seamlessly on all device sizes, from mobile phones to large desktop screens.

## Features

### Navigation & Layout
- **Fixed Header Navigation**: Easy access to different sections with smooth scrolling
- **Responsive Navbar**: Mobile-friendly hamburger menu for smaller devices
- **Search Functionality**: Integrated search box for product discovery
- **Shopping Cart Icon**: Quick access to cart (can be extended with functionality)
- **User Account Icon**: Placeholder for user authentication

### Sections

1. **Home Section**
   - Eye-catching hero banner with welcome message
   - Call-to-action button for quick shopping
   - Background image for visual appeal

2. **About Section**
   - Company history and story
   - Multiple paragraphs describing the coffee shop's journey
   - "Read More" button for extended information
   - Accompanying images

3. **Products Section**
   - Grid layout displaying 6 popular coffee products
   - Product images with hover effects
   - Price display and "Add to Cart" buttons
   - Responsive grid that adapts to screen size

4. **Customers Section**
   - Customer testimonials and reviews
   - 5-star rating system using Box Icons
   - Customer profile images
   - Interactive hover effects

5. **Footer**
   - Multiple footer sections with links and information:
     - Company description with social media links
     - Quick Links (Product, Help & Support, Return Policy, Terms)
     - View Guides (Features, Careers, Blog, Branches)
     - Contact Information (Address, Email, Phone)

## Technologies Used

### Frontend
- **HTML5**: Semantic markup and structure
- **CSS3**: Modern styling with:
  - CSS Grid for layouts
  - Flexbox for component alignment
  - CSS Variables for theme management
  - Media queries for responsive design
  - Smooth scroll behavior
  - Box shadows and border radius effects
- **JavaScript**: Client-side interactivity
  - DOM manipulation
  - Event listeners
  - Toggle classes for menu/search
  - Scroll detection for header shadow effect

### Third-party Libraries
- **Box Icons (CDN)**: Icon library for UI elements
  - Navigation icons
  - Shopping cart icon
  - Search icon
  - User icon
  - Star ratings
  - Social media icons

- **Google Fonts**: Poppins font family for modern typography

## Features Breakdown

### Interactive Elements
- **Mobile Menu Toggle**: Click hamburger icon to show/hide navigation
- **Search Box Toggle**: Click search icon to reveal search input
- **Header Shadow Effect**: Dynamic shadow appears on scroll
- **Hover Effects**: 
  - Navigation links change color on hover
  - Product cards show interactive feedback
  - Customer reviews highlight on hover
  - Social icons change color when hovered

### Responsive Design
- **Mobile First Approach**: Optimized for all screen sizes
- **Breakpoints**:
  - 360px: Extra small devices (phones)
  - 768px: Tablets
  - 991px: Small laptops
  - 1058px: Larger screens
  - 1200px+: Desktop

### Color Scheme
- **Primary Color**: #bc9667 (Warm brown/coffee color)
- **Secondary Color**: #1b1b1b (Dark background)
- **Text Color**: #555 (Dark gray)
- **Background**: #fff (White)
- **Hover Color**: #8a6f4d (Darker brown)

## How to Use

### Getting Started
1. Clone or download this repository
2. Open `index.html` in a web browser
3. No installation or server setup required - it's a standalone static site

### Navigation
- Click on navbar links to jump to different sections
- Use the menu icon on mobile to open the navigation
- Click the search icon to search for products
- Scroll smoothly through the page

### Customization
- Replace images in the `images/` folder with your own
- Update company information in the footer
- Modify product details and prices in the products section
- Edit customer testimonials in the customers section
- Customize colors by editing CSS variables in `style.css`
